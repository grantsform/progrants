# Progrants

This is a collection of mostly grapical (and mosty quickshell based) programs for Grantios.

> Currently we have:

- [ttym](apps/gui/ttym/README.md) — a smart-clock GUI for upcycled ultrabook machines.

- [dedvd](apps/cli/dedvd/README.md) - Help undo the mess my dad made using CD-R/DVD-Rs instead of actual backups.

### Long-term 
I'm using most of these project (sans the cli apps) on glorified appliance devices, single-use thin-clients & upcycled ultrabooks & maybe even eventually a steamdeck. These tend to run either Hyprland or KDE so this stack makes sense atm... but eventually I might port things to [Ozu](https://github.com/metropolisp/ozu) my own gui toolkit x computing platform.
